一、宏观经济环境

1.1 经济增速

指标数据备注

对戈碧迦的影响：中性偏负面

2026年经济目标下调至4.5%-5%，为1991年以来最低，反映决策层对经济增速放缓的理性预期。对于戈碧迦而言：

  • 负面因素：宏观经济增长放缓意味着下游消费电子、汽车等终端需求复苏力度有限，5.47亿营收规模的公司对宏观需求弹性较高
  • 正面因素：政策托底意愿明确，不会系统性衰退，特种功能玻璃（半导体封装、太空光伏）的增量需求受宏观影响相对较小
  • 关键观察：PMI持续位于荣枯线附近徘徊，制造业扩张动力不足，光学玻璃行业的景气度与工业活动高度相关

1.2 消费复苏与内需

  • 2025年零售销售韧性超预期，但2026年年初数据显示消费动能边际放缓
  • 消费者信心修复缓慢，耐用品（含手机等光学玻璃下游终端）换机周期延长
  • 以旧换新政策对消费电子的拉动效应正在递减

对戈碧迦的影响：负面

手机盖板玻璃作为戈碧迦的核心下游之一，直接受制于消费电子终端销售。消费复苏乏力意味着：

  • 智能手机出货量增速受限，盖板玻璃需求增长承压
  • 公司2025年营收已同比下滑3.5%，2026年若消费继续疲软，营收端面临持续压力

北交所当前面临的核心流动性问题：

  • 2025年以来北交所整体成交金额占A股比重不足1%，流动性显著弱于沪深主板
  • 融资端：北交所企业定增、可转债融资渠道相对有限，但政策层持续推进"专精特新"企业融资便利化

  • 融资成本：当前LPR处于历史低位（3.0%），公司债务融资成本较低，有利于财务费用控制
  • 政策催化：若2026年下半年出台北交所专项流动性支持政策（如引入更多做市商、降低投资者门槛），将构成正向催化
  • 风险点：若全球流动性收紧（美联储维持高利率），北向资金可能进一步流出，对北交所流动性造成额外压力

三、产业政策

3.1 半导体国产替代政策

中国持续加大对半导体产业链自主可控的政策支持力度：

  • "十四五"规划（2021-2025）将半导体材料列为重点突破方向，2025年后政策延续性明确
  • 大基金三期（2024年成立，规模超3000亿元）重点投向半导体材料和设备领域
  • 半导体封装玻璃作为芯片封装的关键材料，属于"卡脖子"环节之一，享受政策倾斜

对戈碧迦的影响：正面

戈碧迦的半导体封装玻璃业务直接受益于国产替代浪潮：

  • 公司若能切入国内头部封测企业（如长电科技、通富微电）的供应链，将打开增量市场空间
  • 政策端的资金支持（研发补贴、税收优惠）有助于公司加大半导体级玻璃的研发投入
  • 2025年净利润大幅下滑（-59.6%），产业政策支持有助于公司穿越周期底部

3.2 新材料扶持政策

  • 工信部《重点新材料首批次应用示范指导目录》持续更新，特种光学玻璃位列其中
  • 新材料企业享受15%高新技术企业所得税优惠
  • "专精特新"小巨人企业可获得专项研发资金支持

对戈碧迦的影响：正面

公司作为北交所上市的光学玻璃企业，契合"专精特新"定位，有望持续获得政策支持。

3.3 北交所定位与发展

  • 北交所定位为"服务创新型中小企业的主阵地"，聚焦"专精特新"企业
  • 2025年北交所"深改19条"持续落地，包括优化发行上市制度、完善交易机制等
  • 2026年预期进一步推进转板机制改革，打通与科创板/创业板的通道

对戈碧迦的影响：中性偏正面

  • 若转板政策加速，戈碧迦未来存在转板至科创板/创业板的期权价值
  • 当前北交所估值体系尚未完全理顺，公司估值存在被低估的可能性

四、国际环境与科技脱钩

4.1 中美科技脱钩持续深化

2025-2026年中美科技博弈持续升级：

  • 美国对华半导体出口管制范围扩大，涵盖更多先进制程、半导体材料和设备
  • 光学玻璃和特种玻璃材料作为半导体产业链的上游环节，面临"供应链安全"的紧迫性提升
  • 日本、荷兰等盟友跟随美国对华半导体设备出口管制，进一步强化国产替代逻辑

4.2 供应链自主可控

  • 国内终端厂商（华为、小米、OPPO等）加速推进关键材料的国产替代
  • 光学玻璃领域的国产化率仍有较大提升空间，尤其在高端半导体封装玻璃领域
  • 太空光伏领域作为新兴赛道，供应链自主可控需求迫切

对戈碧迦的影响：正面

科技脱钩对戈碧迦构成结构性的正面催化：

  • 国产替代加速：美国出口管制迫使国内企业寻找本土供应商，戈碧迦作为国内少有的特种光学玻璃制造商，具备"国产替代"标签
  • 客户粘性提升：一旦切入国内头部客户的供应链，客户替换成本高，订单确定性强
  • 太空光伏新机遇：太空光伏产业化加速，特种功能玻璃需求增长，该领域几乎无进口依赖的先例

风险提示：若戈碧迦的核心生产设备（如精密熔炼设备）依赖进口，出口管制可能构成反向制约。

5.1 纳米微晶玻璃（手机盖板）

市场规模：据研究报告测算，2026年纳米微晶玻璃材料消费量达5.6万吨，市场规模超70亿元，近3年CAGR约24%。

  • 竞争格局：苹果2019年首创陶瓷盾玻璃后，华为2022年Mate50系列推出昆仑玻璃，标志着国产微晶玻璃进入主流。常熟佳合、南玻集团、旗滨集团、彩虹集团等正在追赶
  • 戈碧迦的优势：作为国内前三光学玻璃生产商，拥有熔炼技术积累，且已实现大规模量产

风险点：戈碧迦在纳米微晶玻璃链条中处于相对低附加值的熔砖代工环节，终端客户（华为等）的品牌溢价难以直接获取。配方+晶化专利仍在申请中，技术自主性有待提升。

5.2 半导体玻璃基板/载板

行业逻辑：

  • 玻璃基板具有超低平坦度、高热稳定性、低热膨胀系数、高电绝缘性等优点
  • AI算力需求激增，英伟达、英特尔、苹果等海外龙头将玻璃基板作为提升芯片性能的核心方向
  • 扇出型板级封装（FOPLP）使用方型玻璃基板，支持2-10微米线宽的中高阶封装应用

市场规模：2023年全球约71亿美元，预计2028年约84亿美元（CAGR 3.5%）。但在先进封装领域的玻璃基板细分市场增速远超整体。

戈碧迦的机会与挑战：

  • 机会：IPO募投项目包括"特种高清成像光学玻璃材料扩产项目"，新增1632吨/年产能，为向半导体领域延伸奠定产能基础
  • 挑战：半导体玻璃基板对纯度、平整度要求极高，目前由康宁、旭硝子、电气硝子垄断8.5代线以上市场（市占70%+），技术壁垒极高
  • 判断：戈碧迦距离半导体级玻璃基板仍有较大技术差距，短期内难以切入主流供应链，属于中长期战略方向

戈碧迦的相关性：

  • 公司已开发防辐射玻璃、耐高温高压玻璃，这些产品在航天场景有天然适用性
  • 但目前公开信息中尚未有戈碧迦直接切入航天/卫星供应链的明确证据
  • 蓝思科技在UTG领域的布局更成熟，戈碧迦若要进入该领域需跨越技术门槛

二、数据合理性检查

2.1 2023年营收+88%的跳增合理性

数据：2022年营收约4.29亿 --> 2023年8.08亿，+88.3%

验证结论：有合理解释，但需关注以下背景：

  • IPO冲刺效应：公司于2024年3月上市（北交所），2023年为上市前最后一个完整会计年度。从受理到上会历时440余天，远超北交所平均用时，期间存在做大业绩以通过审核的动机。
  • 大客户订单集中释放：2023年纳米微晶玻璃产品（主要客户重庆鑫景特种玻璃）订单集中交付。该客户在2023年贡献了特种功能玻璃收入的主要部分。
  • 产品结构变化：2023年主营构成中"特种功能玻璃"收入5.17亿（占比64%），远高于2024年的1.67亿（占比30%），说明2023年的增长高度依赖纳米微晶玻璃这一单一产品线。
  • IPO问询中已被关注：招股书显示，公司研发费用率低于行业均值，部分客户回款不及预期，应收账款账龄增加——这些都是IPO审核时的关注点。

判定：增长有商业逻辑支撑，但存在"上市前冲量"嫌疑，需结合客户集中度进一步分析。

2.2 2024年营收骤降30%的原因验证

数据：2024年营收5.66亿，同比-29.95%

官方解释（来自业绩快报）： 1. 纳米微晶玻璃产品重要客户重庆鑫景特种玻璃有限公司的订单于4月履行完毕后未获得新订单 2. 其他纳米微晶玻璃客户的增量订单未能弥补重庆鑫景带来的影响 3. 研发费用大幅上升

验证：

  • 该解释合理。2023年特种功能玻璃收入5.17亿（其中纳米微晶玻璃为主要部分），2024年降至1.67亿，下降约3.5亿，基本等于总营收下降额（8.08-5.66=2.42亿），说明该客户的流失确实是主因。
  • 关键风险：公司对重庆鑫景的依赖度过高，单一客户流失即导致营收下降30%，客户集中风险极大。

利润降幅远大于营收降幅的原因分析：

1. 产品结构恶化（主因）： - 2024年特种功能玻璃占比29.6%，毛利率46.4%（高毛利产品拉高整体毛利） - 2025年新拓展的特种功能玻璃客户尚处于导入期，出货量不及预期 - 高毛利的特种功能玻璃销售占比降低，低毛利的光学玻璃占比进一步上升 - 2025H1特种功能玻璃毛利率同比下降17.67个百分点（据研报数据） - 若2024年特种玻璃毛利率为46.4%，则2025年降至约28.7%

3. 股权激励费用摊销： - 2025年实施股权激励计划，首次授予338万股限制性股票（原360万股调整），53名激励对象 - 业绩考核目标：2025年营收较2024年增长10%/20%/30%三档 - 实际2025年营收同比下降3.48%，未达到任何一档考核目标，限制性股票不得解除限售 - 关键问题：尽管股权激励未达成解锁条件，但相关股份支付费用已在2025年摊销，进一步侵蚀了当期利润

  • 假设2025年整体毛利率降至约26%（下降4.6个百分点）
  • 毛利减少约 5.47亿 x 4.6pp = 2,516万
  • 经营杠杆效应（固定费用占比上升）额外影响约500-800万
  • 股权激励摊销影响约300-500万（估算）
  • 合计利润减少约3,300-3,800万，与7081万-2836万=4,245万的降幅基本匹配

判定：净利润降幅大于营收降幅有合理的商业解释，符合产品结构恶化和经营杠杆的逻辑。
