# 航空航天防务行业分析报告

**研究标的**：航宇科技（688669.SH）
**报告日期**：2026-04-07
**数据来源**：iFind宏观行业经济数据库、同花顺资讯

---

## 一、行业概览

### 1.1 行业定义与范围

航空航天防务行业涵盖军用与民用航空器、航天器及其配套系统的研发、制造、维修及服务，产业链覆盖从原材料加工、零部件锻铸、单元体装配到整机总装的全环节。本报告聚焦于航空航天防务行业的军工环节，暨与航宇科技主营务相关的航空发动机环形锻件、航天锻件及燃气轮机锻件赛道。

### 1.2 市场规模与增速

| 指标 | 数据 | 来源 |
|------|------|------|
| 2024年中国航空航天核心产业市场规模 | 突破**2万亿元** | 国泰海通证券 |
| 2025年中国国防预算 | **1.81万亿元**，同比+7.2%（连续三年增速7.2%） | 公开资料 |
| 2025年中国航空装备行业市场规模 | 预计突破**1.8万亿元** | 中研普华 |
| 2023年中国航空航天市场规模 | 超过**1.3万亿元** | 公开资料 |
| 2015-2021年商业航天市场规模复合增长率 | **~20%**（3764→12447亿元） | 中国航天工业质量协会 |
| 2030年商业航天市场规模预测 | **7.8万亿元** | 东吴证券 |
| 2024年卫星导航与位置服务产业产值 | **5773亿元**，同比+7.67% | 公开资料 |
| 2026年低空经济核心产业规模预测 | **突破万亿元** | 中商产业研究院 |

**核心结论**：航空航天防务行业处于长周期高景气阶段，国防预算稳定增长（7.2%）提供基本盘，新兴领域（商业航天、低空经济、卫星互联网）打开成长天花板。

### 1.3 产业链结构图谱

```
【上游】钢铁冶金 / 钛合金 / 高温合金 / 碳纤维复合材料 / 电子元器件
          ↓
【中游】航空锻造（环形锻件/模锻件/自由锻件）
        航空铸造 / 航电系统 / 发动机整机 / 机体结构件
        燃气轮机 / 航天火箭
          ↓
【下游】军用航空（战斗机/运输机/直升机/无人机）
      商用航空（民航客机/发动机）
      航天（运载火箭/卫星/导弹）
      燃气轮机（舰船/工业发电）
```

**航宇科技位于中游核心位置**：主营航空难变形金属材料环形锻件，深度嵌入航空发动机和航天两大高壁垒赛道。

## 二、竞争格局

### 2.1 航空锻件整体竞争格局

航空锻件行业存在**四大壁垒**（技术、资质、客户认可、资金设备），导致国内具备航空锻件生产能力的企业屈指可数，行业呈现"有序竞争"格局。

| 细分赛道 | 主要参与者 | 竞争态势 |
|---------|-----------|---------|
| 机体结构件锻件 | **三角防务**、**中航重机** | 两者主导，工艺路线均以模锻为主 |
| 航空发动机盘件/轴件 | **三角防务**、**中航重机** | 双寡头竞争 |
| 航空发动机环形锻件 | **航宇科技**（688669.SH） | 国内核心龙头，全球稀缺供应商 |
| 高温合金/钛合金精密锻造 | **派克新材**、钢研高纳、兰石超合金 | 细分领域专业竞争 |
| 航天火箭锻件 | 航宇科技、三角防务、派克新材 | 共同竞争 |

**关于航宇科技**：公司是A股市场航空发动机环形锻件的稀缺龙头标的，竞争对手主要为未上市的科研院所（中航工业旗下各专业化工厂）及少量非上市民企。国际对标为美国PCC（Precision Castparts Corp，已被伯克希尔收购）、法国Forge etc。

### 2.2 环形锻件竞争格局详解

航空发动机环形锻件是航宇科技的核心赛道，其关键特征：

- **价值占比**：航空发动机环形锻件约占航空发动机价值的**6%**
- **技术壁垒**：难变形金属材料（高温合金、钛合金）环形锻件成型、高精度环轧工艺，综合检测集成
- **格局特点**：与美国PCC、日本企业等国际巨头直接竞争，同时存在差异化分工
- **航宇科技的优势**：
  - 覆盖国内主流航发型号（长江系列国产商用航空发动机）
  - 进入GE航空、普惠、赛峰、罗罗、普惠等**国际航发巨头**供应链体系
  - 是LEAP-1C（配套C919）的供应商，为高低压涡轮、燃烧室等关键部位提供环形锻件

### 2.3 行业集中度

- 国内航空锻造市场整体CR5（前五大）集中度较高，国有企业（中航重机、三角防务等）与少数民营企业（航宇科技、派克新材）主导
- 2024年国内航空锻铸造市场规模约**250亿元**，中航重机市场份额领先，全市场空间预计达400-600亿元

## 三、上下游分析

### 3.1 上游原材料

| 上游材料 | 主要供应商 | 议价能力 |
|---------|-----------|---------|
| 高温合金 | 抚顺特钢、钢研高纳、图南合金 | 较弱，材料成本占比高 |
| 钛合金 | 宝钛股份、西部超导、兴隆钛业 | 中等，军工认证供应商较稳定 |
| 碳纤维复合材料 | 光威复材、中简科技、中航高科 | 较强，技术壁垒高 |
| 特种钢材 | 东北特钢、重庆特钢等 | 较弱，国产替代充分 |

**关键洞察**：
- 航空锻件原材料成本占营业成本比重较高，稀贵金属（镍、钴、钛）价格波动对毛利率有显著影响
- 国产供应商与国外在稀贵金属循环再生方面存在差距，导致材料成本居高不下
- 航宇科技通过深度绑定上游科研院所和长期锁价合同部分对冲材料成本风险

### 3.2 下游应用与议价能力

| 下游客户类型 | 代表性客户 | 议价能力 |
|------------|-----------|---------|
| 航空发动机整机厂 | 航发动力、GE航空、赛峰、罗罗、普惠 | 较强，型号认证周期长，客户粘性极高 |
| 航天火箭厂商 | 航天科技、九州云箭、星河动力、蓝箭航天 | 较强型号壁垒 |
| 燃气轮机客户 | 工业燃机厂商、舰船配套 | 中等 |
| 商用大飞机 | 中国商飞（配套LEAP-1C） | 中等，商业化初期 |

**关键洞察**：
- 航发赛道下游客户高度集中（航发动力等），订单稳定性强，但议价能力受限
- 国际航发客户（GE、赛峰等）对供应链认证周期长达3-5年，形成强客户粘性
- 商业航天新势力（星河动力、蓝箭航天等）增长快速，提供新收入增量

## 四、行业发展趋势

### 4.1 国防现代化驱动长周期需求

- **装备升级换代**："十四五"期间军机补足中美结构性差异需求持续释放，歼-20、运-20等主力机型年产能力突破200架
- **国防预算刚性增长**：2025年国防预算1.81万亿元，增速7.2%，航空航天装备是装备建设核心方向，装备预算占比提升至35%
- **高超音速飞行器、预警机等高端装备**需求迫切，新型号持续迭代

### 4.2 商业航天：从单点突破到系统集成

- **政策加持**：2026年《政府工作报告》首次将航空航天列为"新兴支柱产业"，明确提出"加快发展卫星互联网，推进商业航天高质量发展"
- **低轨卫星互联网**：中国星网"GW星座"规划发射**超1.2万颗卫星**，带动万亿级产业链需求；2026年国内商业发射占比预计超60%，发射次数突破100次
- **商业航天全产业链垂直整合趋势**：三角防务联合西测测试等成立合资公司，布局"星箭一体化"，从锻件商向整箭整星系统交付转型
- **头部企业资本化加速**：银河航天启动IPO、氦星光联借壳上市，行业集中度持续提升

### 4.3 低空经济：新兴增长引擎

- **2024年中国低空经济核心产业规模突破6000亿元，预计2025年达到1.5万亿元，CAGR超20%**
- **应用场景**：无人机物流（美团、顺丰、京东配送网络日均突破10万单）、eVTOL载人飞行器（亿航、小鹏汇天、峰飞航空主导）
- **基础设施**：5G-A通感一体化网络、低空智联网建设大规模开工，带动通信/导航/监测设备需求

### 4.4 国产大飞机产业化加速

- C919于2024年顺利完成首次跨境商业飞行，机体制造国产化率持续提升
- C919复材占比12%，C929预计复材占比51%，**钛合金和复合材料应用比例提升**推动高端锻件需求
- LEAP-1C发动机配套（航宇科技为关键供应商）持续放量

### 4.5 技术升级趋势

- **精密冷温热锻、超高强钢热成形、等温锻造、超塑性成形**等先进工艺广泛应用
- **整体化、大型化**：大型飞机、宽体客机需求推动更大尺寸、更重结构件发展
- **绿色航空**：航空锂电池、氢燃料电池技术研发投入超1200亿元

## 五、行业风险提示

| 风险类型 | 具体风险 | 影响程度 |
|---------|---------|---------|
| **型号研制风险** | 下游航发型号研制进度不及预期、暂停或取消 | 高 |
| **原材料价格波动** | 高温合金、钛合金、镍等大宗商品价格大幅上涨压缩毛利率 | 中 |
| **军品定价风险** | 军品价格审价机制可能导致产品价格下调 | 中 |
| **国际市场风险** | 国际航发客户订单受地缘政治、出口管制影响 | 中 |
| **商业航天失败率** | 商业火箭发射失败、技术验证不达预期 | 中 |
| **竞争加剧** | 新进入者扩产导致行业产能过剩 | 中 |
| **型号认证周期长** | 航空发动机型号认证周期3-5年，短期业绩释放不确定性高 | 低 |
| **商誉减值风险** | 子公司阶段性亏损及商誉减值影响报表（如航宇科技2025年已受此影响） | 低 |

## 六、核心结论

航空航天防务行业处于**长周期高景气阶段**，国防预算刚性增长（7.2%）提供基本需求保障，商业航天、低空经济、卫星互联网等新兴赛道打开成长天花板。

**航宇科技（688669.SH）** 作为国内航空发动机环形锻件的核心龙头，具备以下核心优势：
1. **技术壁垒高**：掌握航空难变形金属材料环形锻件成型、高精度环轧工艺等关键技术
2. **客户优质**：深度绑定国内航发型号，同时全面进入GE、赛峰、罗罗、普惠等国际供应链
3. **品类齐全**：覆盖航空发动机锻件、航天锻件、燃气轮机锻件完整产品矩阵
4. **全球配套能力**：C919配套LEAP-1C发动机环形锻件，国产大飞机商业化提供长期增量

**建议关注催化剂**：型号研制进展、国际订单释放、商业航天配套订单、募投产能落地节奏。
