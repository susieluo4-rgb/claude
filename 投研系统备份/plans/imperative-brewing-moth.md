# 每日日报定时任务设计方案

## Context

用户希望创建一个每日日报定时任务，每天北京时间 07:00 自动发送。日报需要包含 **10个内容板块**：
1. 全球重要新闻（新增）
2. 全球财经新闻（新增）
3. 隔夜美股综述
4. 贵金属与大宗商品
5. 全球市场概览
6. A股盘前参考
7. 宏观日历
8. 行业热点
9. 个股关注（48家持仓）
10. 公告速递（48家持仓）

**数据源要求**：
- 全球要闻：Reuters + AP News
- 全球财经：Bloomberg + Financial Times
- 其他：ifind MCP + Yahoo Finance API

## 设计方案

### 1. 定时任务配置

| 参数 | 值 |
|------|-----|
| 执行时间 | `0 7 * * *` (北京时间) |
| cron表达式 | `0 7 * * *` + `tz: "Asia/Shanghai"` |

### 2. 发送平台

| 平台 | 状态 | 发送方式 |
|------|------|---------|
| 飞书 | ✅ 已配置 | OpenClaw: `openclaw agent --channel feishu` |
| Discord | ⚠️ 未打通 | 先跳过，完成飞书后再处理 |

### 3. 日报内容结构（共10个板块）

```
## 🌍 全球重要新闻
- 来源: Reuters + AP News + WebSearch
- 内容: 过去24小时全球重大政治/经济/社会事件
- **美伊战争最新情况**（单独板块）
- 数量: **至少5条**

## 💹 全球财经新闻
- 来源: Bloomberg + Financial Times + WebSearch
- 内容: 过去24小时全球重大财经事件
- 数量: **至少5条**

## 📈 隔夜美股综述
- 标普500、纳斯达克、道琼斯收盘
- 恐慌指数 (VIX)

## 🥇🥈 贵金属与大宗商品
- 黄金、白银
- 原油

## 📊 全球市场概览
- 港股ADR
- 日经、欧股

## 🌅 A股盘前参考
- 股指期货 (沪深300、上证50)
- 富时中国A50

## 📅 宏观日历
- 今日重要经济数据发布 (CPI/PPI/PMI/FOMC等)

## 🔥 行业热点
- 每日热门板块/概念
- 来源: ifind trending_news

## 📰 个股关注 (48家)
- 从 Portfolio.xlsx 读取公司列表
- 使用 ifind `search_news` 查询每家公司最新新闻
- **只显示有新闻的公司，无新闻则跳过**
- 每家公司最多显示1-2条新闻

## 📢 公告速递 (48家)
- 使用 ifind `search_notice` 查询最新公告
- **必须显示具体公告内容摘要**
- 显示所有重大公告（业绩预告、分红、重组、监管问询等）
- **不做条数限制**

## 📋 业绩预告
- 数据来源: ifind `get_stock_events` 查询48家公司的业绩披露日期
- 筛选条件: 业绩披露日期 = 今日或明日
- 显示内容: 公司名称、代码、业绩类型（预告/正式财报）、关键数据摘要
- **如无符合条件的公司，则显示"今日/明日无业绩披露"**
```

### 4. 数据获取策略

| 步骤 | 工具/来源 | 说明 |
|------|-----------|------|
| 4.1 | 读取 xlsx | 解析 Portfolio.xlsx 中的48家公司 |
| 4.2 | 全球要闻 | WebSearch: Reuters + AP News |
| 4.3 | 全球财经 | WebSearch: Bloomberg + FT |
| 4.4 | 美股/大宗 | Yahoo Finance API 或 ifind |
| 4.5 | A股/港股 | ifind `get_stock_performance` |
| 4.6 | 宏观数据 | ifind `get_edb_data` |
| 4.7 | 行业热点 | ifind `search_trending_news` |
| 4.8 | 个股新闻 | ifind `search_news` 批量查询48家 |
| 4.9 | 公告查询 | ifind `search_notice` 批量查询48家 |
| 4.10 | 业绩预告 | ifind `get_stock_events` 查询48家业绩披露日期，筛选今日/明日 |

### 5. 发送方式

```bash
openclaw agent \
  --session-id "782328a8-8c4c-4be3-b18c-57ad4ad0ae89" \
  --channel feishu \
  --reply-to "ou_aae8836476a244334c897fb11b9efd1a" \
  --deliver \
  -m "<日报内容>" \
  --json
```
