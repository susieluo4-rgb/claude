# 投研 HTML 报告生成器 v2.0

## 触发条件

当8个Agent分析全部完成后，用户说：
- "生成HTML报告"
- "输出报告"
- "生成v2.0报告"
- "帮我生成报告"

---

## 核心原则

### ⚠️ 数据正确性是最高优先级

**宁德时代案例教训**：
- 09_综合报告.md 中写"净利润722亿"✓，但 04_基本面研究.md 中写"净利润600亿"✗
- 04基本面研究内部也有两个不同净利润（722亿 vs 600亿）
- 原因：没有用统一数据源，各Agent独立生成后未做交叉验证

**规则：iFind MCP 年报数据 = 唯一数据源**，所有财务数字必须以iFind返回为准。

---

## 执行步骤

### 第一步：获取公司股票代码

如果用户只提供了公司名称，先查询股票代码：
```
使用 mcp__hexin-ifind-ds-stock-mcp__search_stocks 查询股票代码
```

---

### 第二步：调用 iFind 获取实际年报数据（必须）

使用以下工具获取实际财务数据：

**A. 财务数据（年报）**
```
mcp__hexin-ifind-ds-stock-mcp__get_stock_financials
query: "{公司名}{股票代码} {最新报告期} 营业收入 归属于母公司所有者的净利润 每股收益 销售毛利率 净资产收益率ROE 经营活动产生的现金流量净额 资产负债率"
```

**B. 估值数据**
```
mcp__hexin-ifind-ds-stock-mcp__get_stock_summary
query: "{公司名}{股票代码} 最新市值 PE PB 股价 机构目标价"
```

**C. 风险指标（可选）**
```
mcp__hexin-ifind-ds-stock-mcp__get_risk_indicators
query: "{公司名} Beta 波动率 VaR"
```

---

从iFind返回的财务数据中，提取以下字段（以2025年年报为例）：

|------|------------|--------|

| 净利润（含少数） | 净利润 | 767.86亿 |
| EPS | 每股收益EPS-基本 | 16.14元 |
| 毛利率 | 销售毛利率 | 26.27% |
| 净利率 | 销售净利率 | 18.12% |
| ROE（加权） | 净资产收益率ROE(加权,公布值) | 24.91% |
| 经营现金流 | 经营活动产生的现金流量净额 | 1332亿 |
| 总股本 | 总股本（亿股） | 44.74亿 |
| 最新股价 | 收盘价 | 386.46元 |
| PE（TTM） | 市盈率(PE,TTM) | 24.6倍 |
| PB | 市净率（PB，MRQ） | 5.3倍 |
| EV/EBITDA | 企业倍数(EV/EBITDA) | 18.1倍 |

### 第四步：生成2026-2028年盈利预测

基于年报实际数据 + 合理假设构建预测表：

**预测模板（填入实际公司数据后使用）**：

| 指标 | 2025年实际 | 2026E | 2027E | 2028E | 假设 |
|------|-----------|--------|--------|--------|------|
| 营收增速 | +17% | +15%~25% | +15%~25% | +10%~20% | 行业增速+公司份额 |
| 归母净利润 | 722亿 | +15%~25% | +15%~25% | +10%~20% | 与营收增速同步 |
| 净利率 | 17% | ±0.5pp | ±0.5pp | ±0.5pp | 规模效应/竞争格局 |
| EPS | 16.14元 | 按净利润/股本 | 同上 | 同上 | 总股本不变 |

**关键假设必须注明**：
- 动力电池/储能/其他业务的收入拆分增速
- 毛利率假设（历史均值/趋势）
- 汇率/大宗商品等外部变量

### 第五步：更新所有 Markdown 文件

检查并修正以下文件中的错误数据（以iFind数据为准）：

| 文件 | 必须含有的关键数据 |
|------|-----------------|
| 04_基本面研究.md | 归母净利润、毛利率、净利率、EPS、ROE、经营现金流 |
| 09_综合报告.md | 与04保持一致的关键数据（股价、PE、净利润） |
| 07_风险评估.md | Beta、波动率、VaR（来自get_risk_indicators） |

**修正规则**：
- 如果Markdown中的数字与iFind数据差异>1%，必须修正
- 标注清楚是"归母净利润"还是"净利润（含少数股东）"，不能混用

### 第六步：生成 v2.0 HTML 报告

**使用模板**：
`/Users/zhuang225/0.Agent 投研总监/reports/template_v2.html`

**填充数据清单**：

#### A. 摘要页必须填充

```
{{STOCK_NAME}}         = 公司简称
{{STOCK_CODE}}         = 股票代码
{{DATE}}               = 今天日期
{{CURRENT_PRICE}}      = iFind最新股价
{{RL_SCORE}}           = 基本面Agent的RL评分总分
{{RL_LEVEL}}           = "良好"/"优秀"等
{{INVESTMENT_RATING}} = 汇总Agent的评级
{{TARGET_PRICE}}       = 目标价区间
{{STOP_LOSS}}          = 止损价
{{RISK_LEVEL}}         = 风险等级
{{RISK_SCORE}}         = 风险评分
{{BETA}}               = iFind Beta
{{VOLATILITY}}         = iFind 波动率
{{INSTITUTIONAL_TARGET}} = 机构一致目标价

// RL五维评分
{{DIM_A_SCORE}} = 8.0
{{DIM_B_SCORE}} = 8.5
...（来自基本面Agent报告）

// KPI Grid
{{APPROVE_COUNT}}      = 给出正面结论的Agent数量
{{MACRO_SIGNAL}}        = 宏观环境一句话
{{INDUSTRY_SIGNAL}}     = 行业一句话
...（8个Agent各自的信号）
```

#### B. 基本面页必须填充（新增2026-2028预测）

在 `<!-- Financial Forecast -->` 注释处插入预测表格：

```html
<!-- Financial Forecast -->
<div class="content-card">
    <div class="card-title"><span class="icon">🔮</span>盈利预测（2026-2028E）</div>
    <p style="font-size:12px;color:var(--text-light);margin-bottom:10px;">
        假设：{详细假设说明}
    </p>
    <table>
        <tr><th>指标</th><th>2025年（实际）</th><th>2026年E</th><th>2027年E</th><th>2028年E</th></tr>
        <tr><td>营收（亿元）</td><td class="number">{实际值}</td><td class="number">{预测}</td>...</tr>
        ...
    </table>
</div>
```

#### C. 估值数据更新

使用iFind实际PE/PB/EV/EBITDA替换模板中的占位符。
