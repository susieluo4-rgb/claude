## Context

2026年4月2日曹操出行业绩交流会后，模型部分假设与管理层指引出现重大偏差，需要系统性调整。主要变化：
- 管理层给出明确的盈利指引（2026年EBIT盈亏平衡、2027年净利润率2-3%），而模型预测持续亏损
- 26年收入增速指引20-25%，模型为26.1%
- 季度与年度数据存在口径不一致

## 调整一：模型假设页 — 盈利能力参数重设

**文件**：模型假设 Sheet

| 参数 | 现状（26E-30E） | 调整后 |
|------|----------------|--------|
| 营收增速 | 26.1% / 25.4% / 20.4% / 18.2% / 15% | **20-25%** / 20% / 18% / 15% / 15%（与指引对齐） |
| 综合毛利率 | 9.8% → 11.7% | 维持（9.4%基准每年+1pct合理） |
| 营业利润 | 持续亏损 | **2026E non-GAAP EBIT 盈亏平衡** |
| 归母净利率 | 持续亏损 | **2027E 2-3%** |

**操作**：
1. 营收增速：26E从26.1%调降至**22%**（区间中值），27E从25.4%调至**20%**，28E维持**18%**，29-30E维持**15%**
2. 销售费用率：当前6%/5.5%/5.2%/5.0%/4.8%，可适当降低（聚合平台佣金稳定在6-7%，规模效应显现）
3. 管理费用率：维持（11%/10%/9.5%/9%/8.5%）
4. 有效税率：25A为0，26E维持0，27E从8%调整至实际税率（若27年转盈，实际税率约8-10%）

## 调整二：利润表 — 2026E EBIT盈亏平衡、2027E净利润率2-3%

**文件**：利润表 Sheet

根据管理层指引反推关键参数：

**2026E non-GAAP EBIT 盈亏平衡**
- revenue_26E = 201.9 × 1.22 ≈ **246亿元**
- 毛利率假设：9.8% → 毛利润 ≈ 24.1亿
- 费用假设：销售费用率6% → 14.8亿；管理费用率11% → 27亿；研发费用率2.8% → 6.9亿
- 营业利润 ≈ 24.1 - 14.8 - 27 - 6.9 = **-24.6亿**（仍然亏损）
- **矛盾**：管理层指引EBIT盈亏平衡，但按现有费用率模型无法实现
- **需要调整**：销售费用率需从6%降至约**4-5%**才能实现EBIT盈亏平衡
  - 原因：聚合平台佣金6-7%已是稳定水平，规模效应主要体现在降低佣金率而非继续压缩

**2027E non-GAAP净利润率 2-3%**
- revenue_27E ≈ 246 × 1.20 ≈ **295亿元**
- 若净利润率3% → 净利润 ≈ **8.9亿元**
- 反推：295 × 3% = 8.85亿；但要同时满足EBIT盈亏平衡
- 结论：需要**毛利率提升至约11%+**，且费用率进一步压缩

**实际操作**：
1. 在"模型假设"页新增两行：
   - `2026E Non-GAAP EBIT盈亏平衡（管理层指引）`
   - `2027E Non-GAAP净利润率2-3%（管理层指引）`
2. 同步调整利润表Sheet中2026E/2027E的营业利润和净利润预测
3. 在摘要Sheet同步更新2026E/2027E的营业利润/净利润/净利率数字

## 调整四：现金流量表 — 校核

**现状**：模型中2026E经营现金流为**-11.5亿**，但管理层明确表示"25年经营性现金流和自由现金流均已转正"

**调整**：
- 2025A经营现金流：模型6亿（符合），需确认FCF
- 2026E经营现金流：若EBIT盈亏平衡，+D&A约7亿 → 经营现金流应显著改善
- 建议：2026E经营现金流调整为**正值**（约3-5亿），与管理层"25年已转正并大幅增加"的口径一致

## 调整五：季度数据修复（低优先级）

**问题**：Q4 2025营业利润+1.5亿，但年报全年为-4.633亿，Q1-Q3合计约-4亿，三者不一致。

**处理**：
- 以年报数字为准（审计数据更可靠）
- 季度数据表可能存在口径差异（IFRS vs non-GAAP），暂时标记`待核实`，不强行拉平
