# 宏观环境分析 — 圣邦股份

> 报告日期：2026-04-07 | 数据来源：iFind EDB、公开资讯

---

## 一、全球宏观环境

### 1.1 全球经济增长

2025年全球经济呈现分化格局。美国GDP同比增速录得5.0%，受益于AI产业投资拉动和财政扩张，经济韧性超预期。欧洲经济复苏乏力，受能源成本高企和制造业外迁影响，增速维持在1%左右。全球半导体行业在AI算力需求驱动下强劲反弹，2025年全球半导体销售额达到7917亿美元，较2024年的6305亿美元同比增长25.6%，创历史新高。

### 1.2 美联储利率政策

美联储自2025年下半年开启降息周期，联邦基金目标利率从2025年初的4.50%逐步下调：

| 时间 | 联邦基金目标利率 |
|------|-----------------|
| 2025年1月 | 4.50% |
| 2025年9月 | 4.25% |
| 2025年10月 | 4.00% |
| 2025年12月 | 3.75% |
| 2026年3月 | 3.75%（维持不变） |

截至2026年3月，美联储累计降息75bp，但降息节奏趋缓。市场预期2026年仍有1-2次降息空间，利率终点或落在3.25%-3.50%区间。宽松货币环境整体利好科技股估值修复，对半导体板块形成支撑。

### 1.3 中美科技竞争

中美科技博弈持续升级，半导体领域仍是核心焦点：

- **出口管制层层加码**：美国持续更新实体清单，对先进制程设备（光刻机、刻蚀机）、HBM存储芯片、AI训练芯片的对华出口限制进一步收紧。2025-2026年，荷兰ASML、日本东京电子等对华销售均受到更严格审查。
- **供应链重构加速**：全球半导体供应链从"效率优先"转向"安全优先"，美国推动芯片制造回流（CHIPS法案持续落地），中国加速本土替代。
- **对圣邦股份的双重影响**：短期内出口管制限制了部分高端模拟芯片的工艺迭代路径；但中长期来看，国产替代需求显著增加，下游客户主动切换至国内供应商的意愿空前强烈。

## 二、中国宏观环境

### 2.1 GDP增速

2025年中国GDP不变价增速为4.68%，略低于2024年的5.0%目标，经济增长动能有所放缓。2026年政府工作报告将GDP增速目标设定在4.5%-5.0%区间，强调"以科技创新引领新质生产力"。

### 2.2 货币政策

中国央行维持宽松货币政策基调，2025年5月LPR迎来一次降息：

| 指标 | 2025年初 | 2025年5月起 | 当前（2026年3月） |
|------|---------|------------|------------------|
| 1年期LPR | 3.10% | 3.00% | 3.00% |
| 5年期LPR | 3.60% | 3.50% | 3.50% |
| MLF 1年期 | 2.00% | -- | -- |

- **M2增速**：2025年全年M2同比增长8.5%，2026年预测进一步升至9.0%，流动性保持充裕。
- **社融增速**：2025年社会融资规模存量同比增长8.3%，信用扩张温和。
- 央行强调"适度宽松"的货币政策，降准降息仍有空间，为实体经济和科技创新提供低成本资金支持。

### 2.3 财政政策

2026年财政政策延续积极基调：

- 赤字率预计维持在3.5%-4.0%的较高水平，专项债额度继续扩大。
- "以旧换新"政策延续并扩围，对消费电子、汽车等圣邦下游核心领域形成直接拉动。
- 科技创新领域财政支出持续增长，半导体、人工智能等新兴产业获重点倾斜。

### 2.4 PMI走势

制造业PMI在过去15个月整体围绕荣枯线波动，呈现弱复苏态势：

| 月份 | 制造业PMI |
|------|----------|
| 2025年1月 | 49.1 |
| 2025年3月 | 50.5 |
| 2025年6月 | 49.7 |
| 2025年9月 | 49.8 |
| 2025年12月 | 50.1 |
| 2026年1月 | 49.3 |
| 2026年2月 | 49.0 |
| 2026年3月 | **50.4** |

2026年3月PMI回升至50.4，表明制造业景气度季节性修复，春季复工复产带动生产和新订单回暖。但整体来看，PMI中枢仍在50附近徘徊，制造业需求端恢复力度偏弱。

### 2.5 价格环境

- **CPI**：2025年全年CPI指数为100.0（同比持平），通缩压力犹存。
- **PPI**：2025年全年PPI同比-2.6%，工业品价格持续下行，反映产能过剩和需求不足。

低通胀环境一方面为货币政策宽松提供空间，另一方面也意味着下游需求疲软，对圣邦股份的终端定价和毛利率构成一定压力。

## 三、半导体行业政策环境

### 3.1 国家大基金三期持续发力

国家集成电路产业投资基金（大基金）三期自2024年设立以来，累计注册资本超过3400亿元，三期合计大基金总规模突破6200亿元。2026年重点投向：

- **先进制造与封装**：中芯国际、华虹公司、通富微电等产能扩张；
- **半导体设备国产化**：北方华创、中微公司、拓荆科技等获重点支持；
- **存储芯片**：长鑫科技上市预期成为自主可控里程碑。

地方层面同步加码：上海IC基金三期注册资本大幅增至60亿元（增幅超11倍），显示地方政府对半导体产业的资金支持力度空前。

### 3.2 芯片自主可控政策

"自主可控"已上升为国家战略层面的长期方向：

- 2026年政府工作报告再次强调"加快关键核心技术攻关"，半导体被列为"新质生产力"的核心领域。
- 国产化率考核制度逐步落地，关键领域芯片采购优先使用国产方案。
- AI算力芯片、存储芯片、模拟芯片等细分赛道均被纳入重点支持范围。
- 行业研报指出，2026年国内晶圆厂扩产加速，半导体设备订单能见度直达年底，国产化率有望进一步提升。

### 3.3 出口管制影响

美国对华半导体出口管制对不同细分领域影响差异明显：

- **高端数字芯片（GPU/AI芯片）**：受限最严，倒逼国产GPU厂商（摩尔线程、沐曦等）加速追赶。
- **模拟芯片**：直接制裁压力相对有限，因模拟芯片制程以成熟节点（28nm及以上）为主，不在最严格管制范围内。但供应链安全意识提升，下游客户加速导入国产模拟芯片。
- **设备与材料**：受限程度持续加深，但国内设备厂商在刻蚀、薄膜沉积、清洗等环节国产替代进展显著。

## 四、对圣邦股份的影响分析

### 4.1 国产替代：核心增长驱动力

圣邦股份作为国内模拟芯片龙头，直接受益于国产替代浪潮：

- **供应链安全需求**：在中美科技博弈持续背景下，下游终端厂商（华为、比亚迪、小米等）将供应链安全提升至战略优先级，主动切换至国产模拟芯片供应商。
- **政策红利**：大基金及地方产业基金对半导体全产业链的资金支持，有望间接降低圣邦的研发和产能建设成本。
- **模拟芯片赛道优势**：模拟芯片以成熟制程为主，受出口管制直接冲击较小，且产品生命周期长、客户粘性高，国产替代壁垒相对可控。

### 4.2 下游需求展望

| 下游领域 | 需求趋势 | 关键驱动因素 |
|---------|---------|------------|
| **消费电子** | 弱复苏 | "以旧换新"政策拉动、AI手机/AI PC渗透率提升，但整体换机周期延长 |
| **工业控制** | 稳中偏弱 | PMI在荣枯线附近波动，工业自动化升级需求持续但增速放缓 |
| **汽车电子** | 结构性增长 | 新能源汽车渗透率持续提升（2025年超50%），单车模拟芯片用量显著增长，智能驾驶对高精度ADC/电源管理芯片需求旺盛 |
| **AI/服务器** | 高景气 | AI算力基础设施建设带动电源管理芯片需求，2026年服务器成为存储第一大下游应用 |

### 4.3 宏观环境综合评估

| 维度 | 评级 | 说明 |
|------|------|------|
| 货币政策 | **偏利好** | LPR处历史低位，流动性充裕，利好科技股估值 |
| 财政政策 | **利好** | 专项债+以旧换新政策直接拉动下游需求 |
| 产业政策 | **强利好** | 大基金三期+自主可控政策，模拟芯片赛道受益确定性高 |
| 外部环境 | **双刃剑** | 出口管制加速国产替代，但也限制技术引进和国际合作 |
| 需求端 | **分化** | 汽车电子/AI高景气，消费电子/工业弱复苏 |

## 五、宏观风险提示

1. **中美科技脱钩超预期**：若出口管制进一步扩展至成熟制程或模拟芯片领域，可能对圣邦的晶圆代工和IP授权渠道产生直接影响。
2. **国内经济复苏不及预期**：PMI持续在荣枯线下方运行、PPI深度通缩，可能导致下游客户去库存周期延长，压制模拟芯片出货量和ASP。
3. **美联储降息节奏放缓**：若美国经济持续强劲导致降息暂停甚至转向，可能引发全球风险偏好回落，对A股科技板块估值形成压制。
4. **行业产能过剩风险**：国内晶圆厂大规模扩产可能导致成熟制程产能供给过剩，模拟芯片行业竞争加剧，毛利率承压。
5. **地缘政治黑天鹅**：台海局势、俄乌冲突等地缘事件可能对全球半导体供应链造成突发冲击。
6. **消费复苏不确定性**：CPI零增长、居民消费意愿偏弱，消费电子换机需求可能持续低迷。
7. **汇率波动**：人民币兑美元汇率波动可能影响圣邦的进口原材料成本及海外收入。
