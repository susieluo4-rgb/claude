## 一、公司概述

### 1.1 基本信息

| 项目 | 内容 |
|------|------|
| 公司全称 | 贵州航宇科技发展股份有限公司 |
| 上市地 | A股科创板 |
| 上市日期 | 2021-07-05 |
| 主营业务 | 航空难变形金属材料环形锻件研发、生产和销售 |
| 核心产品 | 航空发动机环形锻件（占收入~80%） |
| 可比公司 | 中航重机、宝鼎科技、无锡派克新材、西安三角防务 |

### 1.2 股权结构与治理

- 公司为民营控股，总股本约1.91亿股（截至2025年末）
- 2025年完成可转债转股（航宇转债，代码118050），累计转股10,571股
- 股东减持：累计减持66,033股已回购股份（占总股本0.03%），主要用于偿还贷款
- 子公司：成都成航发通用动力设备有限公司（2024年10月并表）、贵州钜航表面处理技术有限公司

### 1.3 核心投资逻辑

> 国内航空环形锻件龙头，在手订单近60亿元保障未来2-3年业绩确定性；2025年营收20.34亿元（+12.64%）境内外双增长，核心净利润2.11亿元（+11.48%）；海外市场（GE、赛峰、罗罗、普惠）持续放量，全球市场份额目标从高个位数提升至20%-30%；2030年收入目标50亿元，其中两机业务占比80%。

### 1.4 核心竞争优势

1. **技术领先**：自研NSP（近净成形）技术，节省原材料20%-50%，获机翼航空全球供应商大会Innovation奖
2. **客户壁垒**：通过GE、赛峰、罗罗、普惠等全球航发巨头认证，供应商资质壁垒高
3. **全球格局**：全球航空锻件"两超三强"格局（两超：PCC、Howmet；三强：Firth Rixson、Frisa、Forgital及航宇科技），公司份额提升趋势确定
4. **产业链延伸**：从锻件向精密加工、单元体组件交付延伸（精加工毛利率50%-60%），价值量提升30%-40%

### 3.2 利润表关键科目分析

**收入端：**
- 2024年收入18.05亿元（-14.2%），主要受国内客户交付节奏调整影响
- 2025年收入20.34亿元（+12.64%），境内外双增长，核心为海外航发持续放量

**利润端：**
- 2025年归母净利润1.86亿元（-1.44%），主要受子公司阶段性亏损及商誉减值影响
  - 成都成航发亏损926万元
  - 贵州钜航表面处理亏损1064万元
- 扣除非经常性损益后核心净利润2.11亿元（+11.48%），更能反映主业韧性

**盈利能力：**
- 毛利率28.03%（+1.2pp），展现成本控制能力
- 净利率10.4%（+1.6pp），利润韧性优于收入
- ROE 10.82%（加权），仍有提升空间

### 3.3 季度财务表现（2025年Q3）

- Q3单季度收入超过6亿元，同比+46%，环比+20%以上
- Q3单季度利润增长30%+，为近2-3年最强劲季度
- Q3扣非后净利率9.1%（全年最高，Q2为8.38%，Q1为7.62%），经营质量显著提升

**客户结构：**

- GE：占海外收入20%左右，推进较快

### 6.2 核电业务

- **发展历程**：2021年开始参与钠冷熔盐堆关键配套，参与早、程度深、产品品类多
- **增长潜力**：若小型熔盐堆批产并大规模应用，将成为公司大的增量业务
- **其他核电**：布局其他核电资质，取得后核电业务有望更上台阶

### 6.3 半导体设备零部件

- **进展**：已交付样件并进入验证阶段，小批订单下发中
- **目标市场**：全球每年百亿人民币规模
- **2026年收入预期**：大几百万-1000万元
- **长期目标**：争取10%-20%市场份额（国内小几十亿市场）

- **市场份额**：锻件环节约60%-70%
- **毛利率**：30%-35%（来料加工模式）
- **收入展望**：未来几年有望达到小几个亿规模

## 八、投资亮点与风险提示

### 8.1 投资亮点

1. **军工航空赛道长期确定性高**：C919/ARJ21国产大飞机放量，商用航发市场复苏
2. **在手订单充足**：约60亿元在手订单保障未来2-3年业绩
3. **海外市场高速增长**：2025年海外收入9.11亿元（+19.35%），全球市场份额目标20%-30%
4. **产业链延伸提升附加值**：精加工、组件交付模式使毛利率从28%向40%+迈进
5. **NSP技术提升竞争力**：节省原材料20%-50%，已获全球航空龙头认证
6. **多元新兴业务布局**：燃气轮机（2030年目标10-15亿）、核电、半导体设备等

### 8.2 风险提示

| 风险类型 | 具体风险 | 影响程度 |
|---------|---------|---------|
| 交付节奏风险 | 国内客户交付节奏调整可能导致收入波动 | 高 |
| 子公司亏损风险 | 成都成航发、贵州钜航持续亏损可能影响利润 | 中 |
| 估值偏高风险 | PE 56×处于历史高位，含军工概念溢价 | 中 |
| 型号放量节奏 | 军工型号从研发到批量需要长周期 | 中 |
| 原材料价格风险 | 高温合金、钛合金价格波动影响成本 | 低 |
| 汇率风险 | 海外收入占比提升，人民币汇率波动影响业绩 | 低 |
| 国际贸易风险 | 海外客户集中度高，地缘政治变化可能影响供应 | 低 |
