# Plan: 敏实集团财务模型 — 全面重构为公式联动架构

## Context

v2版本的敏实集团模型虽然科目对齐了CATL格式，但核心问题是**所有数据都是Python计算后写入的值**，而非Excel公式跨Sheet联动。这导致：
1. 修改假设（如某个业务增速）后，三张报表不会自动更新
2. 摘要的历史/预测分两块而非同行合并
3. 利润率/YOY都是写死的数值，不是公式

用户要求全部重构为CATL式的公式联动模型（业务拆分→模型假设→利润表→资产负债表/现金流量表）。

---

## 核心理念

**数据流向**：
```
业务拆分(蓝色输入) → 模型假设(汇总+手动设置) → 利润表(公式引用假设)
                                        ↓
                              资产负债表/现金流量表(引用利润表)
```

### 1. 摘要 Sheet — 历史/预测合并到同行

**改动前**：历史指标一块（H列），预测指标另一块（E列后）
**改动后**：同一行内，历史列（H列填值）+ 预测列（E列填公式）

示例：
| | 2022A | 2023A | ... | 2025A | 2026E | 2027E | 2028E | 2029E |
|--|-------|-------|-----|-------|-------|-------|-------|-------|
| 营业收入 | 值 | 值 | ... | 值 | =利润表!2026E | =利润表!2027E | ... | ... |
| YoY% | =历史计算 | =历史计算 | ... | =历史计算 | =公式 | =公式 | ... | ... |

### 2. 业务拆分 Sheet — 可调输入驱动中心

参考CATL格式，**蓝色可改输入区**：

**输入区（蓝色，手动可调）**：
- 每行业务：收入增速（%）、毛利率（%）
- 示例：
  | 业务板块 | 2025A增速 | 2025A毛利率 | 2026E增速 | 2026E毛利率 | 2027E增速 | 2027E毛利率 |
  |---------|----------|------------|----------|------------|----------|------------|
  | 电池盒 | 实际值 | 实际值 | **20%** | **28%** | **18%** | **28%** |
  | 外饰件/结构件 | 实际值 | 实际值 | **8%** | **28%** | **7%** | **28%** |
  | 智能前脸/其他 | 实际值 | 实际值 | **15%** | **30%** | **12%** | **30%** |

**计算区（绿色/黑色公式，自动计算）**：
- 每行业务：收入（=上年*(1+增速)）、毛利（=收入*(1-毛利率)）
- 合计行：总收入（=SUM各业务）、总毛利（=SUM各业务毛利）、**加权平均毛利率**（=总毛利/总收入）、**加权增速**（=总收入/上年总收入-1）

**关键输出（引用到模型假设）**：
- `E44` = 总收入 → 利润表引用
- `E46` = 加权毛利率 → 模型假设引用
- `E47` = 加权增速 → 模型假设引用

---

### 3. 模型假设 Sheet — 联动控制中心

**两部分组成**：

**A. 自动汇总（引用业务拆分，绿色）**：
- 营收增长率：=业务拆分!加权增速行
- 毛利率：=业务拆分!加权毛利率行

**B. 手动可调（蓝色）**：
- 研发费用率（%）
- 销售费用率（%）
- 管理费用率（%）
- 有效税率（%）
- 归母/合并净利比例（%）
- D&A金额（亿）
- Capex金额（亿）
- 股息支付率（%）
- 营运资本参数：应收账款天数、存货天数、应付账款天数

**关键：所有值都是实际Excel公式字符串**，如：
```python
ws.cell(row=5, col=2, "=业务拆分!E47")  # 营收增速
ws.cell(row=7, col=2, "=业务拆分!E46")  # 毛利率
ws.cell(row=8, col=2, 5.8)              # 研发费用率（蓝色可改）
```

### 4. 利润表 Sheet — 公式驱动

**历史列（H）：** 填入Python计算的实际值
**预测列（E）：** 使用Excel公式引用模型假设

示例公式：
```
# 营业收入
I5: =H5*(1+模型假设!B5/100)
J5: =I5*(1+模型假设!C5/100)

# 营业成本
I7: =ROUND(I5*(1-模型假设!B7/100),1)

# 毛利润
I8: =I5-I7

# 研发费用
I13: =ROUND(I5*模型假设!B9/100,1)

# 营业利润
I18: =I8-I11-I12-I13-I15+I16

# EBITDA
I20: =I18+模型假设!B15

# 净利润
I22: =ROUND(I18*(1-模型假设!B12/100),1)

# 归母净利润
I23: =ROUND(I22*模型假设!B13/100,1)
```

**利润率/YOY行：**
```
# 毛利率
I9: =I8/I5

# YoY
I6: =I5/H5-1

# 营业利润率
I19: =I18/I5

# 归母净利YoY
I24: =I23/G23-1
```

### 5. 资产负债表 Sheet — 公式联动

**关键链接公式**：
```
# 应收账款 = 营收 * 应收账款天数/365
I6: =利润表!I5*模型假设!B18/365

# 存货 = 营业成本 * 存货天数/365
I8: =利润表!I7*模型假设!B19/365

# 应付账款 = 营业成本 * 应付账款天数/365
I22: =利润表!I7*模型假设!B20/365

# 货币资金 = 现金流量表期末余额
I5: =现金流量表!I27

# 固定资产 = 上年+Capex-D&A
I13: =资产负债表!H13+模型假设!B16-模型假设!B15

# 股东权益 = 上年+归母净利-分红
I35: =H35+利润表!I23*(1-模型假设!B22/100)

# 资产负债率
I41: =I33/I19
```

---

### 6. 现金流量表 Sheet — 公式联动

```
# 经营现金流 = 净利润 + D&A - 营运资本变动
I4: =利润表!I23+模型假设!B15-(I6-I5)*模型假设!B18/365

# Capex
I5: =-模型假设!B16

# 自由现金流
I6: =I4+I5

# 期末货币资金 = 资产负债表货币资金
I12: =资产负债表!I5
```
